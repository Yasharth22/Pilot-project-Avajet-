# Pilot-Project-1-

Avajet CAMO Data Management System is a sponsored final-year engineering project developed to streamline aircraft maintenance and operations (CAMO) activities. The system enables efficient data management for aircraft, materials, and maintenance records through a secure and user-friendly interface.

Key Features:
 Dashboard – Displays aircraft data summaries and maintenance status at a glance.
 Administration Module – Handles user roles, permissions, and configuration.
 My Aircraft – Manage aircraft details, maintenance records, and operational data.
 My Materials – Tracks spare parts, consumables, and inventory data.
 Database Integration – Real-time connectivity for storing and retrieving aircraft-related data.

Tech Stack:
Frontend: React.js, HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDb Atlas
Version Control: Git & GitHub

Purpose:
Developed as part of the final-year sponsored project, this software is designed to assist CAMO departments in managing large-scale aircraft operational data efficiently, improving decision-making and data traceability.
